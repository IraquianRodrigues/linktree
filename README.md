Peça+Açaí - Site Oficial
Este é o site oficial do Peça+Açaí, uma açaiteria que oferece a melhor experiência em sabor e atendimento. O projeto tem como objetivo proporcionar aos clientes um acesso fácil e direto ao cardápio, localização, e formas de contato com a loja.

Tecnologias Usadas
Este projeto foi desenvolvido utilizando as seguintes tecnologias:

HTML5: Para estruturar o conteúdo do site.
CSS3: Para o design e estilo visual da página.
JavaScript: Para interatividade, incluindo o modal para exibir o cardápio digital.
Ionicons: Biblioteca de ícones para adicionar ícones modernos e responsivos no site.
Google Fonts: Utilizado para carregar a fonte "Inter" e melhorar a legibilidade do conteúdo.

Funcionalidades
Cardápio Digital: Acesso fácil ao cardápio da açaiteria diretamente pelo site, com a possibilidade de visualizá-lo em formato de imagem em modal.
Localização: Link direto para o Google Maps com a localização exata da loja.
WhatsApp: Contato fácil com a açaiteria via WhatsApp para pedidos.
Redes Sociais: Links para as redes sociais do fundador, Iraquian Rodrigues, para acompanhar as novidades e interagir.
Responsividade: O layout foi projetado para ser responsivo e funcionar bem em diferentes dispositivos.

Como Usar
Acesse o site: Ao abrir o site, você verá informações sobre o fundador e a frase "Quem prova, sempre pede mais!".
Cardápio Digital: Ao clicar no link do cardápio digital, o modal será aberto, exibindo a imagem do cardápio.
Entre em Contato: Clique no link do WhatsApp para enviar uma mensagem diretamente para a açaiteria.
Saiba a Localização: O link para o Google Maps abrirá a localização exata da loja.
